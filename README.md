# corrupted.io

Personal tech blog / security research log. Built with [Jekyll](https://jekyllrb.com/), served via GitHub Pages.

## Stack

- **Static site:** Jekyll 4.x on GitHub Pages
- **Theme:** Custom dark-only cyberpunk — cyan/magenta neon palette, CSS-only cityscape + sliced sun, monospace headings, proportional body text
- **Plugins:** jekyll-seo-tag, jekyll-feed, jekyll-sitemap, jekyll-paginate, jekyll-redirect-from
- **Fonts:** System font stack (no external requests)
- **Analytics:** None

## Local development

```bash
bundle install
bundle exec jekyll serve
```
